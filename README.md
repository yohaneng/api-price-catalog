World of APIs - Pricing Catalog
This project is a comprehensive directory of over 400 APIs. It is designed to help users find, compare, and understand the pricing structures of various digital services.

Key Features
Searchable Database: Easily find APIs by name or function.

Categorized by Domain: Filters for AI/ML, Finance, IoT, and more.

Automated Updates: Prices are checked and updated daily using AI.

Transparency Audit: Focus on identifying clear vs. obfuscated pricing models.

Technical Details
Data Source: dataset_400.json

Automation: GitHub Actions + Python Scraper
